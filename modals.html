InfoSecu

Précédent Suivant

InfoSecu est une application intranet de Philips Lighting. Philips Lighting (VSG) est un centre de distribution régionale qui est capable de fournir des services dédiés pour des clients qui recherchent des solutions logistiques spécifiques.

Elle a pour objectif primaire de contraindre les chauffeurs et visiteurs entrants sur le site de prendre connaissance des consignes de sécurité. Mais aussi de faciliter le travail du personnel au poste de garde.

  • Mission : Développer une application qui contraint la visualisation des consignes et qui permet de numériser les enregistrements des entrant afin de créer une impression.
  • Date de développement : Mai-Juin 2017 et Janvier-Février 2018
  • Entreprise cliente: Philips Lighting
  • Compétences mises en oeuvre :Rédaction d'une documentation technique, VB.net avec module métier,"pilotage" de word avec Powershell, jQuery (méthodes js AJAX), Framework .NET de Microsoft, Framework CSS Bootstrap 3.0, Mise en production
  • Categorie: Application web intranet de prévention
  • Documentation: InfoSecu-Documentation.pdf
  • Dernière remise en production: Bordereau_de_remise_en_production.pdf
  • Github: https://github.com/JonathanCrt/infoSecu

FaceCast

Cette application permet de basculer une partie des traitements d’une part sur une application Web et d’autre part sur une application mobile Android. La première de type Intranet est destinée au « backoffice » (non accessible aux utilisateurs finaux) et la deuxième au « frontoffice » (pour les figurants).

  • Mission: Concevoir et développer une application intranet afin de faciliter le travail du personnel DRH de l'agence.

  • Categorie: Application web NodeJS
  • Documentation: faceCast-documetation-technique.pdf

GéoWorld-Android

GéoWorld est une application Android qui à pour objectif de faire découvrir des données géographiques à partir d'un fichier Json.

Explications : A partir d'un fichier Json nommé world.json et d'un dossier d'images (contenant le drapeau de chaque pays) on affiche les pays dans une ListView Android. La ListView n'affiche que les informations sommaires du pays (nom, capitale, drapeau …). Lorsque l'on clique sur un pays, des informations plus précises s'affichent dans une AlertDialog.

  • Mission: Développer une application Android qui permet à ses utilisateurs de découvrir des données géopolitiques de manière simple et intuitive.
  • Date de développement: Avril 2017
  • Projet de fin de semestre
  • Compétences mises en oeuvres : Android, Java orienté objet, Layout
  • Catégorie: Application Android SDK
